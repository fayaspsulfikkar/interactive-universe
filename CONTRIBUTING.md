# Contributing to Interactive Universe Explorer

First off, thank you for considering contributing to Interactive Universe! It's people like you that make Interactive Universe such a great project.

## Code of Conduct

By participating in this project, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

### Reporting Bugs
This section guides you through submitting a bug report. Following these guidelines helps maintainers properly understand your report, reproduce the behavior, and find related reports.

- **Use the bug report template**: Head over to the Issues tab and use the provided Bug Report template.
- **Explain the problem thoroughly**: Include details about your browser, device, and steps to reproduce.
- **Provide screenshots if applicable**: 3D rendering issues are often best explained visually.

### Suggesting Enhancements
This section guides you through submitting an enhancement suggestion.

- **Check existing issues**: Before creating an enhancement suggestion, please check if one already exists.
- **Use the feature request template**: Clearly outline your suggestion, the problem it solves, and potential alternatives.

### Pull Requests
1. **Fork the repo** and create your branch from `main`.
2. **If you've added code that should be tested, add tests.** (If applicable).
3. **If you've changed APIs, update the documentation.**
4. **Ensure the test suite passes.**
5. **Make sure your code lints.**
6. **Submit that pull request!**

### Styleguides

#### Git Commit Messages
- Use the present tense ("Add feature" not "Added feature")
- Use the imperative mood ("Move cursor to..." not "Moves cursor to...")
- Limit the first line to 72 characters or less
- Reference issues and pull requests liberally after the first line

#### JavaScript / React Styleguide
- All JavaScript code must conform to the project's ESLint configuration.
- We prefer functional components and React Hooks.
- Three.js components should ideally be broken down into composable, reusable parts using `@react-three/fiber`.
